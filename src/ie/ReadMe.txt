     A HREF++     Ver3.1      
------------------------------
 2010/11/23

* 概要
  - Internet Explorer用右クリック拡張ソフトです。ハイパーリンク化されていない
    URLを選択して実行すると、新しいウィンドウが開いて飛ぶことができます。
  - ttp から始まる URLや、誤URLも自動で補完します。
  - 詳しい説明は以下を参照下さい。
    http://irts.jp/program/browserplugin/ahrefplusplus/

* 使い方
  - URLを含む文字列を選択し、右クリックから「A HREF++」を実行して下さい。
    文字列に含まれる複数のURLを新しいウィンドウで開くことができます。
  - Shiftを押しながら実行すると、URLに ime.nu を付加します。
  - Ctrlを押しながら実行すると、選択範囲を強制的に一つのURLとみなします。
  -「A HREF++」の動作を以下のページでお試し下さい。
    http://irts.jp/program/browserplugin/ahrefplusplus/#practice

* インストール/アンインストール
  - A_HREF_plusplus_SetUp.vbs をダブルクリックし、インストールする際は「はい」
    アンインストールする際は「いいえ」を選択して下さい。
  - インストール後は Internet Explorer を再起動する必要があります。

* 動作環境
  - InternetExplorer 6/7/8
  - Windows 7/Vista/XP

* 参考情報
  - セキュリティソフトによってはインストール時に警告を出す場合があります。
    危険な事はしていないので、このプログラムを許可して下さい。
  - 複数のURLを開く場合、ブラウザやツールバーのポップアップブロック機能が発動
    する場合があります。必要に応じてポップアップブロック機能をオフにして下さい。
  - インターネットオプションのタブの設定（IE7/8）で「ポップアップを開く方法を
    Internet Explorer で自動的に判定する」あるいは「常に新しいタブでポップ
    アップを開く」にすると、A HREF++ で開いた URL はすべてタブで開かれます。

* About
  - Right click extention for Internet Explorer.
  - Open the URLs included in selected string even if the URL is incomplete.
    (Ex. "ttp://example.com/" "http:/example.com/" )
  - See this page.
    http://irts.jp/program/browserplugin/ahrefplusplus/

* Usage
  - Select URLs(not Hyperlinked), then rignt click and execute "A HREF++".
  - Here is the practice page for "A HREF++".
    http://irts.jp/program/browserplugin/ahrefplusplus/#practice

* Install/Uninstall
  - Double click "A_HREF_plusplus_SetUp.vbs".

* Environment
  - InternetExplorer 6/7/8
  - Windows 7/Vista/XP

* Tips
  - In case some security software alert you when you install "A HREF++",
    please permit this program. 
  - When you open many URLs by "A HREF++" at once, Pop-up Blocker may block
    opening windows. Please turn off Pop-up Blocker as necessary.
  - I recommend you to change Tabs Setting of "Internet Options" (IE7/8),
    "Let InternetExplorer decide how pop-ups should open" or "Always open
    pop-ups in a new tab".
